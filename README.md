## Executive Summary

**NEXUS** is a decentralized Information Finance (InfoFi) terminal deployed exclusively within the Telegram Mini App (TMA) ecosystem.

NEXUS turns internet culture, social sentiment, and data veracity into a liquid, tradable asset class. We allow Telegram's 900+ million users to take binary Long/Short positions on real-time social event outcomes through frictionless onboarding and native TON wallet integration.

## Key Features

### 🔹 Native Web3 Integration
* **TON Connect UI V2:** Immediate synchronization with native TON wallets (Tonkeeper, Telegram Wallet, etc.) for non-custodial asset settlement.
* **Multi-Chain Abstraction:** A simulated backend framework designed to aggregate assets across TON, Solana, and Ethereum into a single unified view.

### 🔹 High-Frequency InfoFi Feed
* **Predictive Markets:** Long/Short contracts on social media trends, public sentiment, and on-chain data anomalies.
* **Dynamic Odds:** An algorithmic model (simulated via AI House) that adjusts odds in real-time based on trade volume and information velocity.

### 🔹 "Zero-Friction" UX
* **Single-File SPA:** Built using vanilla JavaScript (ES6+), optimized for instant load times and native feel inside Telegram.
* **Telegram WebApp SDK:** Tight integration with Telegram themes, expanded viewport, and back-button logic.

## Core Technology Stack

This repository showcases the complete frontend MVP architecture within a **single `index.html` file**. This approach ensures rapid deployment, zero-cost hosting, and zero friction for external evaluators.

* **HTML5/CSS3:** Professional Web3, dark-mode design optimized for mobile portrait orientation.
* **JavaScript (ES6+):** Advanced client-side logic handling state management, SPA routing, and simulated live tickers without external frameworks.
* **Telegram SDK:** Native communication with the Telegram application.
* **TON Connect SDK (via CDN):** Direct wallet connectivity without `npm` setup steps.

## Deployment & Development Guide

This is a production-ready repository that requires zero technical setup.

### How to Deploy (to update links for investors):

1.  Create a public repository named `nexus-app`.
2.  Upload `index.html` to the repository.
3.  Go to **Settings > Pages** and enable GitHub Pages on the main branch.
4.  You now have a live HTTPS URL (e.g., `https://yourusername.github.io/nexus-app`).
5.  Link this URL to your Telegram Bot via `@BotFather` using the `/newapp` command.
